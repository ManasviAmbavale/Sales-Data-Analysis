# Sales Data Analysis using Python

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, profitability, and performance across different regions and product categories.

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset used in this project is the **Sample Superstore Dataset**, which contains information about sales, profit, discounts, product categories, and regions.

## Analysis Performed
The following analysis was performed in this project:

- Data loading and exploration
- Data cleaning and checking for missing values
- Sales analysis by category
- Profit analysis by region
- Discount vs Profit relationship
- Top performing product categories
- Data visualization using charts and graphs

## Key Insights
- Technology category generated the highest sales.
- Some regions produced high sales but lower profits.
- Higher discounts negatively impacted profitability.
- A few product categories contribute most of the revenue.

## Conclusion
Sales performance varies across categories and regions. Technology products generated the highest revenue, while higher discounts often reduced profitability.
